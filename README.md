📋 Sobre o Projeto
Site de portfólio criado para apresentar minhas habilidades, projetos e informações de contato de forma clara e profissional. O foco foi construir uma aplicação leve, sem frameworks, com código organizado e boa experiência visual.
🔗 Ver online →

✨ Funcionalidades

Animações de entrada — elementos surgem suavemente ao rolar a página com IntersectionObserver
Menu com destaque ativo — o link da seção atual é destacado automaticamente durante a rolagem
Seções completas — Hero, Habilidades, Projetos, Formação e Contato
Links de contato diretos — WhatsApp, E-mail, GitHub e LinkedIn
Design responsivo — adaptado para dispositivos móveis e desktop
Zero dependências — sem frameworks ou bibliotecas externas


🛠️ Tecnologias Utilizadas
Tecnologia Usei HTML5 e strutura semântica das páginas CSS3 e stilização, animações e layout responsivo JavaScript e PagesHospedagem gratuita
